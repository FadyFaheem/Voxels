Settings

Advanced device settings

Device

A friendly name for your device. This will be shown in the status screen.

WiFi Network

The name of your WiFi network. Leave blank to keep current setting.

WiFi password. Leave blank to keep current password. Device will reconnect after saving.

Time & Date

Select your timezone. The clock widget will display time in this timezone.

Display

Adjust the font size for all widgets. Changes will apply immediately.

⚠️ Danger Zone

Reset all settings and start fresh. This will erase all saved configuration including WiFi credentials, device name, and widget settings. The device will restart.
